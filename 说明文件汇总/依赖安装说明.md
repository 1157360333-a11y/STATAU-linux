# ============================================================================
# STATAU 依赖包安装说明
# ============================================================================

## 核心依赖（必须安装）

所有依赖已在 requirements.txt 中列出，运行以下命令安装：

```bash
pip install -r requirements.txt
```

## 环境切换功能所需的额外依赖（可选）

为了使用环境切换功能，建议安装以下两个包：

### 1. python-dotenv（推荐）
用于自动加载 .env 文件中的环境变量

```bash
pip install python-dotenv
```

**作用**：
- 自动读取 .env 文件中的配置
- 无需手动设置系统环境变量
- 更方便的开发体验

**如果不安装**：
- 需要手动设置系统环境变量（如 `export RUN_MODE=local`）
- 或者直接修改代码中的配置类

### 2. flask-session（服务器版必需）
用于服务器版的文件系统 Session 存储

```bash
pip install flask-session
```

**作用**：
- 在服务器版（RUN_MODE=server）中启用文件系统 Session
- 防止服务器重启后用户掉线
- 支持多进程部署

**如果不安装**：
- 服务器版会回退到内存 Session（默认）
- 服务器重启后所有用户需要重新登录
- 多进程部署时 Session 不共享

## 快速安装命令

```bash
# 安装所有依赖（包括可选依赖）
pip install -r requirements.txt
pip install python-dotenv flask-session

# 或者一次性安装
pip install -r requirements.txt python-dotenv flask-session
```

## 验证安装

```bash
# 检查是否安装成功
pip show python-dotenv
pip show flask-session
```

## 更新 requirements.txt

如果需要将这两个包添加到 requirements.txt：

```bash
# 方式1：手动添加到 requirements.txt
echo python-dotenv >> requirements.txt
echo flask-session >> requirements.txt

# 方式2：导出当前环境的所有包
pip freeze > requirements.txt
```
